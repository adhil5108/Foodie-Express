# FoodieExpress Plus (JSON Server)
- React + Vite + Tailwind (responsive)
- JSON Server for users & products
- Search + Category filter + Sort (Menu)
- Per-user cart saved on server:  /users/:id
- Protected routes: /cart, /checkout

## How to Run app
npm install
npm run dev
```

## Start JSON Server (separate terminal)
json-server --watch db.json --port 1000
`
``
Endpoints:
- http://localhost:1000/users
- http://localhost:1000/products
